Группа пользователей по умолчанию — группа пользователей, в которую пользователь попадает после регистрации на сайте.

Полная документация

Пользователь-гость — здесь можно задать пользователя из списка, чьи права будут использоваться для незарегистрированных пользователей. Эта настройка используется, если по каким-то причинам был удален исходный пользователь-гость. В других случаях использовать не рекомендуется.

Регистрация пользователей без активации — при включенной опции, аккаунт будет активирован автоматически после регистрации пользователя.
